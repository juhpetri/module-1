# Atividade 2 - Classe Pedido

> Crie uma classe `Pedido` com os seguintes atributos:
> - `numero` (String)
> - `valorTotal` (Double)
> - `pago` (Boolean)
> - `cliente` (Cliente)

Implemente:
- Construtor completo
- Métodos `get` e `set`
- Um método `resumo()` que imprima:
  "Pedido [numero] - Pago: Sim/Não - Total: R$ [valor] - Cliente: [nome] - Email: [email]"
